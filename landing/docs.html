Quick Start

# Convert HTML to PDF (no API key needed, rate-limited)
curl -X POST https://pdfapi.uhadev.com/api/v1/convert/public \
-H "Content-Type: application/json" \
-d '{ "html": "<h1>Hello World</h1>" }' \
-o output.pdf
Free API key: Sign in at pdfapi.uhadev.com to get an API key with a 10,000 conversion limit — no credit card required.

Authentication

Authenticated requests use a Bearer token in the Authorization header. Get your key by signing in on the landing page.

Authorization: Bearer pdfping_<your_api_key>

Without a key, use the /public endpoint which is limited to 50 conversions per day per IP.

Endpoints

POST /api/v1/convert Authenticated conversion

Requires a valid API key in the Authorization header. Supports all options including custom page format, margins, and landscape. Counts toward your API key usage limit.

POST /api/v1/convert/public Public conversion (rate-limited)

No API key required. Limited to 50 conversions per day per IP address.

Request Parameters

Send a JSON body with Content-Type: application/json.

Parameter Type Default Description
html string — HTML string to render. Provide this or url.
url string — URL to render. Provide this or html.
options.format string A4 Page format. One of: A3, A4, A5, Letter, Legal, Tabloid.
options.landscape boolean false Render in landscape orientation.
options.printBackground boolean true Include CSS background colors and images.
options.margin string 0mm Page margin (CSS units, e.g. 10mm, 0.5in). Applied to all sides.
options.wait number 1 Seconds to wait after page load before generating PDF. Increase for JS-rendered content.

Example request body

{
"html": "<h1>Invoice</h1><p>Hello</p>",
"options": {
"format": "A4",
"landscape": false,
"margin": "10mm"
}
}

Responses & Errors

Success (200)

Returns the PDF binary with Content-Type: application/pdf. The Content-Disposition header is set to inline so browsers display the PDF directly.

Error responses

Status Body Cause
400 { "error": "..." } Missing or invalid request body (no html or url).
401 { "error": "..." } Missing Authorization header on authenticated endpoint.
403 { "error": "..." } Invalid API key.
429 { "error": "..." } Rate limit exceeded on the public endpoint.
500 { "error": "...", "detail": "..." } Conversion failed (invalid HTML, unreachable URL, browser error).

Code Examples

curl

# Authenticated — with options
curl -X POST https://pdfapi.uhadev.com/api/v1/convert \
-H "Authorization: Bearer pdfping_YOUR_KEY" \
-H "Content-Type: application/json" \
-d '{
"html": "<h1>Hello World</h1>",
"options": { "format": "A4", "margin": "10mm" }
}' \
-o output.pdf

JavaScript (fetch)

// Node.js or browser
const res = await fetch("https://pdfapi.uhadev.com/api/v1/convert", {
method: "POST",
headers: {
"Content-Type": "application/json",
"Authorization": "Bearer pdfping_YOUR_KEY",
},
body: JSON.stringify({
html: "<h1>Hello World</h1>",
}),
});

// Save to file (Node.js)
const buffer = Buffer.from(await res.arrayBuffer());
require("fs").writeFileSync("output.pdf", buffer);

Python

# pip install requests
import requests

res = requests.post(
"https://pdfapi.uhadev.com/api/v1/convert",
headers={
"Authorization": "Bearer pdfping_YOUR_KEY",
"Content-Type": "application/json",
},
json={
"html": "<h1>Hello World</h1>",
"options": { "format": "A4" },
},
)

with open("output.pdf", "wb") as f:
f.write(res.content)
